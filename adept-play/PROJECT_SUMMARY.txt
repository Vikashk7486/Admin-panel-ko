WORKFLOW EXAMPLE:
================================================================================

USER WORKFLOW:
1. User registers/logs in
2. Browses tournaments on homepage
3. Clicks "Join Now" (entry fee deducted from wallet)
4. Views tournament in "My Tournaments"
5. When admin sets tournament to "Live", user sees Room ID & Password
6. After tournament, admin declares winner
7. Winner receives prize in wallet automatically
8. User sees result in "Completed" tab

ADMIN WORKFLOW:
1. Admin logs in
2. Creates new tournament with details
3. Users join the tournament
4. Admin sets Room ID & Password
5. Admin changes status to "Live"
6. After tournament ends, admin selects winner
7. Clicks "Declare Winner & Distribute Prize"
8. Prize automatically added to winner's wallet
9. Tournament marked as "Completed"
